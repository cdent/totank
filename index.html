To Tank

While Tank can host TiddlyWiki (Classic and Five) the mechanism is as yet not well tested and is very different from how things are done on TiddlySpace. If you are primarly a TiddlyWiki user on TiddlySpace it is probably best to experiment with how things work at Tank before doing a migration of any complex gathering of TiddlyWiki plugins.

If, however, you are mostly interested in getting your tiddler content somewhere you can work with it, and you like Tank, this migration tool may be useful to you.

You should be sure to learn about the permissions settings on tanks. They are different from the public/private setup on TiddlySpace.

Tiddlers you send from here to Tank will clobber tiddlers of the same name in the target tank.

This is not ready yet!

This space provides a simple tool for migrating the content of a space to one or two tanks at Tank, yet another service for hosting tiddlers.

In order to use the tool you need to already have an account at Tank (you can sign up with a GitHub, Facebook or Google account), have generated an auth key and created the one or more target tanks you wish to migrate to.

Once you are set up at Tank you can then include this space in the space you wish to migrate and visit totank.html.
